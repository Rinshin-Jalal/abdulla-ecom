
# Vite Color Scheme

The Vite application uses a sophisticated and premium color palette designed for a modern, high-end e-commerce experience.

## Primary Colors

| Name | Variable | Hex Code | Description |
| :--- | :--- | :--- | :--- |
| **Canvas** | `--canvas` | `#FAF9F6` | The primary background color. An off-white, editorial beige. |
| **Ink** | `--ink` | `#1C1C1C` | The primary text color. A deep, elegant black. |
| **Accent** | `--accent` | `#3A4D39` | A deep forest green used for primary actions, success states, and branding. |
| **Clay** | `--clay` | `#C08261` | An earthy terracotta used for secondary accents, badges, and warm highlights. |

## UI & Surface Colors

| Name | Variable | Hex Code | Description |
| :--- | :--- | :--- | :--- |
| **Surface** | `--surface` | `#F2F0EA` | Used for cards, containers, and secondary background layers. |
| **Border** | `--border` | `#E5E2D9` | A subtle sand-colored border used for low-contrast structure. |

## Usage Guidelines

- **Typography**: Titles typically use `--ink` or `--accent`. Subtitles often use `--ink` with 40-60% opacity.
- **Buttons**: The primary button style uses `--accent` with white text. Secondary labels use `--clay`.
- **Interactions**: Hovers often shift toward `--accent` or increase depth using soft shadows.
- **Micro-interactions**: Subtle transitions use `cubic-bezier(0.19, 1, 0.22, 1)` for a smooth, expensive feel.
